# Chain

Chain is being prepared as an AI toolkit. This repo now keeps the docs surface focused while preserving reusable package boundaries for future toolkit work.

## What is in this repo now

- A modern Next.js and Fumadocs documentation app.
- Shared TypeScript configuration.
- A small UI package ready to grow into toolkit components.
- PNPM catalog and Syncpack checks to keep dependency versions coherent.

## Development

```bash
pnpm install
pnpm dev
```

The docs app runs on port `3001` by default.
